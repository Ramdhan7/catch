# catch
Skrip khusus crack wilayah indonesia dengan kecepatan clone yang luar biasa


### Install :
````bash
$ pkg update && pkg upgrade 
$ pkg install python2 
$ pkg install git 
$ pkg install bs4
$ pkg install nano
$ pkg install ruby
$ pip2 install mechanize 
$ pip2 install requests 
$ git clone https://github.com/Ramdhan7/catch
````
### Run Script :
````bash
$ cd catch
$ ls
$ python2 catch.py


````
